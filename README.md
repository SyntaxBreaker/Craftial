## Technologies Used

- Typescript
- Mongoose
- Auth0


## Getting started
`git clone https://github.com/iaincollins/nextjs-starter.git`

`npm install`

`Create a .env file and enter your MONGODB_URI, for example: MONGODB_URI='your_MONGODB_URI'`

`npm run dev.`
